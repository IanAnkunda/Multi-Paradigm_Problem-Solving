This project computes the mean, median, and mode of a list of integers using three different programming paradigms:
- C (Procedural)
- OCaml (Functional)
- Python (Object-Oriented)

---

## How to Run

### C (Procedural)
cd Multi-Paradigm_Problem-Solving/c  
sudo apt update
sudo apt install build-essential -y
gcc stats.c -o stats  
./stats

### OCaml (Functional)
cd Multi-Paradigm_Problem-Solving/ocaml 
sudo apt update
sudo apt install ocaml -y 
ocaml stats.ml

### Python (Object-Oriented)
cd Multi-Paradigm_Problem-Solving/python  
python3 statistics_calculator.py
